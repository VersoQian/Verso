Before play the Game you need:

1. Import DOTween plugin
https://assetstore.unity.com/packages/tools/animation/dotween-hotween-v2-27676

2. Install iOS 14 Advertising Support package
https://docs.unity3d.com/Packages/com.unity.ads.ios-support@1.0/manual/index.html

3. Install Vibration plugin
https://github.com/BenoitFreslon/Vibration

4. Set up TextMesh Pro
https://docs.unity3d.com/Packages/com.unity.textmeshpro@3.0/manual/index.html
IMPORTANT! Find TMP Settings. Default Sprite Asset section. Replace Path: Sprite Assets/ to Sprites/

5. Set up IAP, Ad Providers.
https://www.notion.so/Common-Documentation-18bce6b4237c80ca9ad3fc5857a804e4

6. Set up Newtonsoft Json plugin
https://github.com/applejag/Newtonsoft.Json-for-Unity/wiki/Install-official-via-UPM

7. Set up Input System (v1.14.2)
https://www.notion.so/Common-Documentation-18bce6b4237c80ca9ad3fc5857a804e4
